实用工具推荐

Visual Studio Code：像WebStrom，功能肯定比文本编辑器丰富，还自带 服务器，可能新手会比较喜欢用这种方便的工具。但是有了nodejs，我们前端可以自己搭建服务器 来测试我们的项目。编辑器轻量，快速，功能也越来越丰富。再加上一些编辑器是使用前端语言 Javascript写的，最后选择了文本编辑器。

比较火的编辑器有notepad，sublime，atom等， VS Code是出来没多久的。它吸收了前面编辑器的优秀功能，还增加了自己独特的功能，速度还比 atom快，这就是我为什么一路走来最终选择了VS Code。

目前我认为VS Code比较好用的几个功能： 内置Git，如果公司使用Git，或者自己的项目通过Git管理，那么用起来会非常方便。 调试功能，我还没有发现哪个编辑器有这个功能。 丰富的插件，通过插件可以扩展很多功能。可以支持很多语言，增加很多主题。

Ngrok：当你做了一款应用，能用本地的服务器跑起来，但只能自己访问，并不能实际使用。 ngrok能将本地的端口映射到互联网上。这样通过互联网就能访问到你本地的局域网服务器。

去ngrok官网，将ngrok下载下来，双击运行，可以看它的很多命令提示，最简单的方法 就是输入`ngrok http 8080`，然后回车。这种方式会生成一个随机的地址，那个地址就可以通过互联 网访问。但是随机的地址使用起来很不方便，所以可以买它的自定义域名的服务。但是需要使用国外的 银行卡。

百度sunny ngrok你会发现新大陆。根据我的了解，他们是自己花钱有了自定义 域名，然后给我们免费使用。我们只需要去他的网站上注册一个账号，到里面去设置自己的通道，就可以 完成免费固定域名的功能。
